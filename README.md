Бушмина Ольга Маратовна

Аналитик данных с практическим опытом в e-commerce. 
На предыдущей позиции отвечала за анализ продаж, управление ассортиментом и эффективность рекламных кампаний, используя данные из ERP/CRM систем (МойСклад, Webasyst).

За последний год целенаправленно развивала навыки в аналитике, прошла курс «Аналитик данных» (Яндекс.Практикум) и выполнила ряд проектов по исследовательскому анализу данных, A/B-тестированию и прогнозированию.

Мой стек для решения задач:
● SQL: Построение витрин, сложные запросы с CTE и оконными функциями.
● Python: Pandas, NumPy, Matplotlib, Seaborn, SciPy для EDA, статистики и проверки гипотез.
● BI: Tableau, Power BI, DataLens для создания интерактивных дашбордов.

Готова применять аналитический подход и технические навыки для решения ваших бизнес-задач.
